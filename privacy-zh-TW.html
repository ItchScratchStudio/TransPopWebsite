隱私權政策

最後更新：2026年1月

引言

在 SwiftTrans，我們高度重視您的隱私。本隱私權政策說明了當您使用我們的 macOS 應用程式時，我們如何處理您的資料。

資料收集與使用

SwiftTrans 的設計理念是隱私優先。我們不直接收集個人資訊。您的翻譯歷史和設定儲存在您的裝置本機。

翻譯服務

為了提供翻譯功能，您提交的文字會傳送到您選擇的翻譯服務提供商：

  • Google 翻譯：文字會傳送到 Google 的翻譯 API。請參閱 Google 隱私權政策。
  • 必應翻譯：文字會傳送到微軟的必應翻譯服務。請參閱 微軟隱私權聲明。
  • OpenAI / 自訂 API：如果您設定了 OpenAI 或其他 API 端點，資料將根據您的設定傳送到這些服務。請參閱相應 API 提供商的隱私權政策（例如 OpenAI 隱私權政策）。

除了為翻譯目的而必要地傳輸給這些第三方提供商外，SwiftTrans 不會在任何外部伺服器上儲存您提交的文字。

本機儲存

  • 翻譯歷史：儲存在您的 Mac 本機。您可以隨時在應用程式設定中清除此歷史記錄。
  • API 金鑰：如果您提供自己的 API 金鑰（例如 OpenAI），它們會安全儲存在您的 macOS 鑰匙圈中。
  • 偏好設定：應用程式設定（主題、語言等）儲存在標準 macOS 使用者預設設定中。

當機回報

我們可能會收集匿名當機日誌，以協助我們提高應用程式的穩定性。這些日誌不包含個人使用者資料或翻譯內容。
