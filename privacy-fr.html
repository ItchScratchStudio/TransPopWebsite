Politique de Confidentialité

Dernière mise à jour : Janvier 2026

Introduction

Chez SwiftTrans, nous priorisons votre confidentialité. Cette Politique de Confidentialité explique comment nous gérons vos données lorsque vous utilisez notre application macOS.

Collecte et Utilisation des Données

SwiftTrans est conçu avec la confidentialité comme priorité. Nous ne collectons pas d'informations personnelles directement. Votre historique de traductions et vos paramètres sont stockés localement sur votre appareil.

Services de Traduction

Pour fournir la fonctionnalité de traduction, le texte que vous soumettez est envoyé au fournisseur de traduction que vous sélectionnez :

  • Google Translate : Le texte est envoyé à l'API de traduction de Google. Veuillez consulter la Politique de Confidentialité de Google.
  • Bing Translator : Le texte est envoyé au service Bing Translator de Microsoft. Veuillez consulter la Déclaration de Confidentialité de Microsoft.
  • OpenAI / APIs Personnalisées : Si vous configurez OpenAI ou d'autres points de terminaison API, les données sont envoyées à ces services selon votre configuration. Veuillez consulter la politique de confidentialité du fournisseur d'API respectif (par exemple, Politique de Confidentialité d'OpenAI).

SwiftTrans ne stocke pas votre texte soumis sur des serveurs externes autres que la transmission nécessaire à ces fournisseurs tiers dans le but de la traduction.

Stockage Local

  • Historique des Traductions : Stocké localement sur votre Mac. Vous pouvez effacer cet historique à tout moment dans les paramètres de l'application.
  • Clés API : Si vous fournissez vos propres clés API (par exemple, pour OpenAI), elles sont stockées en toute sécurité dans votre Trousseau macOS.
  • Préférences : Les paramètres de l'application (thème, langue, etc.) sont stockés dans les Préférences Utilisateur standard de macOS.

Rapports de Plantage

Nous pouvons collecter des journaux de plantage anonymes pour nous aider à améliorer la stabilité de l'application. Ces journaux ne contiennent pas de données personnelles d'utilisateur ni de contenu de traduction.
