隐私政策

最后更新：2026年1月

引言

在 SwiftTrans，我们高度重视您的隐私。本隐私政策说明了当您使用我们的 macOS 应用程序时，我们如何处理您的数据。

数据收集与使用

SwiftTrans 的设计理念是隐私优先。我们不直接收集个人信息。您的翻译历史和设置存储在您的设备本地。

翻译服务

为了提供翻译功能，您提交的文本会发送到您选择的翻译服务提供商：

  • Google 翻译：文本会发送到 Google 的翻译 API。请参阅 Google 隐私政策。
  • 必应翻译：文本会发送到微软的必应翻译服务。请参阅 微软隐私声明。
  • OpenAI / 自定义 API：如果您配置了 OpenAI 或其他 API 端点，数据将根据您的配置发送到这些服务。请参阅相应 API 提供商的隐私政策（例如 OpenAI 隐私政策）。

除了为翻译目的而必要地传输给这些第三方提供商外，SwiftTrans 不会在任何外部服务器上存储您提交的文本。

本地存储

  • 翻译历史：存储在您的 Mac 本地。您可以随时在应用设置中清除此历史记录。
  • API 密钥：如果您提供自己的 API 密钥（例如 OpenAI），它们会安全存储在您的 macOS 钥匙串中。
  • 偏好设置：应用设置（主题、语言等）存储在标准 macOS 用户默认设置中。

崩溃报告

我们可能会收集匿名崩溃日志，以帮助我们提高应用程序的稳定性。这些日志不包含个人用户数据或翻译内容。
