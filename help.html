Help & User Guide

Overview

SwiftTrans is your elegant translation companion for macOS. Whether you need quick translations while reading, writing, or working, SwiftTrans is designed to fit seamlessly into your workflow.

Getting Started

Installation

Download SwiftTrans from the Mac App Store and launch it. The app will appear in your menu bar and you can access it anytime.

First Translation

Simply paste or type your text into the input field, select your target language, and SwiftTrans will instantly translate it for you.

Key Features

Translation Providers

SwiftTrans supports multiple translation services to give you the best results:

  • Google Translate - Fast, accurate translations with automatic language detection
  • Bing Translator - Microsoft's powerful translation engine
  • OpenAI & Custom APIs - Connect your own API for advanced translations with AI models

Switch between providers in Settings to find the one that works best for your needs.

Mini Mode

Access SwiftTrans's compact Mini Mode for quick translations that don't interrupt your work. Perfect for translating short snippets while reading or writing.

  • Trigger Mini Mode with the global keyboard shortcut ⌘+C+C
  • Lightweight and always accessible
  • Stays on top of other windows when needed

Translation History

All your translations are automatically saved to history, making it easy to find and reuse previous translations.

  • Search through your translation history
  • Copy past translations instantly
  • Set a maximum number of saved items in Settings (default: 100)
  • Clear history anytime for privacy

Subscription Plans

SwiftTrans offers both Free and Pro plans:

  • Free Plan - Access core translation features with basic functionality
  • Pro Plan - Unlock advanced features including unlimited history, priority support, and more

Upgrade to Pro anytime from the Subscription view within the app.

Settings & Customization

Appearance

  • Theme - Choose between Light, Dark, or System appearance
  • Language - Change the app's interface language to your preference

Behavior

  • Close Action - Choose what happens when you close the window (minimize to menu bar, quit, or ask each time)

Privacy & Security

  • API Keys - If using custom APIs, your keys are stored securely in macOS Keychain
  • Local Storage - Translation history is stored locally on your Mac
  • Updates - Enable automatic update checks to stay current

Tips & Tricks

  • Use ⌘+C+C to access SwiftTrans's Mini Mode instantly from any app
  • Toggle between translation providers to compare results
  • Enable text-to-speech to hear your translations pronounced
  • Favorite frequently used language pairs for quick access
