Ayuda y Guía de Usuario

Descripción General

SwiftTrans es tu elegante compañero de traducción para macOS. Ya sea que necesites traducciones rápidas mientras lees, escribes o trabajas, SwiftTrans está diseñado para integrarse perfectamente en tu flujo de trabajo.

Primeros Pasos

Instalación

Descarga SwiftTrans desde Mac App Store e inícialo. La aplicación aparecerá en tu barra de menú y podrás acceder a ella en cualquier momento.

Primera Traducción

Simplemente pega o escribe tu texto en el campo de entrada, selecciona tu idioma de destino y SwiftTrans lo traducirá instantáneamente.

Funciones Principales

Proveedores de Traducción

SwiftTrans soporta múltiples servicios de traducción para darte los mejores resultados:

  • Apple Translation (Sin conexión) - En macOS 15+, traduce completamente en tu dispositivo sin necesidad de internet. Compatible con 20 idiomas.
  • Google Translate - Traducciones rápidas y precisas con detección automática de idioma. Compatible con más de 120 idiomas.
  • Bing Translator - El potente motor de traducción de Microsoft
  • OpenAI y APIs Personalizadas - Conecta tu propia API para traducciones avanzadas con modelos de IA

Cambia entre proveedores en Configuración para encontrar el que mejor funcione para tus necesidades.

Modo Mini

Accede al compacto Modo Mini de SwiftTrans para traducciones rápidas que no interrumpen tu trabajo. Perfecto para traducir fragmentos cortos mientras lees o escribes.

  • Activa el Modo Mini con el atajo de teclado global ⌘+C+C
  • Ligero y siempre accesible
  • Se mantiene sobre otras ventanas cuando es necesario

Historial de Traducciones

Todas tus traducciones se guardan automáticamente en el historial, facilitando encontrar y reutilizar traducciones anteriores.

  • Busca en tu historial de traducciones
  • Copia traducciones pasadas instantáneamente
  • Establece un número máximo de elementos guardados en Configuración (predeterminado: 100)
  • Borra el historial en cualquier momento por privacidad

Gratuito & Pro

SwiftTrans ofrece planes Gratuito y Pro:

  • Gratuito - Accede a funciones de traducción básicas con Google Translate y Bing
  • Pro (Compra única) - Desbloquea traducciones ilimitadas, soporte de API personalizada y todas las funciones premium con una sola compra — sin suscripción necesaria

Actualiza a Pro en cualquier momento desde la aplicación.

Configuración y Personalización

Apariencia

  • Tema - Elige entre apariencia Clara, Oscura o del Sistema
  • Idioma - Cambia el idioma de la interfaz de la aplicación

Comportamiento

  • Acción de Cierre - Elige qué sucede cuando cierras la ventana (minimizar a la barra de menú, salir o preguntar cada vez)

Privacidad y Seguridad

  • Claves API - Si usas APIs personalizadas, tus claves se almacenan de forma segura en el Llavero de macOS
  • Almacenamiento Local - El historial de traducciones se almacena localmente en tu Mac
  • Actualizaciones - Habilita la verificación automática de actualizaciones para mantenerte actualizado

Consejos y Trucos

  • Usa ⌘+C+C para acceder al Modo Mini de SwiftTrans instantáneamente desde cualquier aplicación
  • Alterna entre proveedores de traducción para comparar resultados
  • Habilita texto a voz para escuchar la pronunciación de tus traducciones
  • Marca como favoritos los pares de idiomas que uses frecuentemente para acceso rápido
