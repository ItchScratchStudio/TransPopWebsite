幫助與使用者指南

概述

SwiftTrans 是您在 macOS 上的優雅翻譯夥伴。無論是閱讀、寫作還是工作時需要快速翻譯，SwiftTrans 都能無縫融入您的工作流程。

快速開始

安裝

從 Mac App Store 下載 SwiftTrans 並啟動。應用程式將出現在選單列中，您可以隨時存取。

首次翻譯

只需將文字貼上或輸入到輸入框中，選擇目標語言，SwiftTrans 即刻為您翻譯。

主要功能

翻譯服務

SwiftTrans 支援多種翻譯服務，為您提供最佳結果：

  • Google 翻譯 - 快速準確的翻譯，支援自動語言偵測
  • 必應翻譯 - 微軟強大的翻譯引擎
  • OpenAI 與自訂 API - 連接您自己的 API，使用 AI 模型進行進階翻譯

在設定中切換翻譯服務，找到最適合您需求的選項。

迷你模式

使用 SwiftTrans 的小巧迷你模式進行快速翻譯，不會打斷您的工作。非常適合在閱讀或寫作時翻譯短文字。

  • 使用全域鍵盤快捷鍵 ⌘+C+C 觸發迷你模式
  • 輕量級，隨時可用
  • 需要時可保持在其他視窗上方

翻譯歷史

所有翻譯都會自動儲存到歷史記錄中，方便您查找和重用之前的翻譯。

  • 搜尋您的翻譯歷史
  • 即時複製過去的翻譯
  • 在設定中設定儲存項目的最大數量（預設：100）
  • 隨時清除歷史記錄以保護隱私

訂閱方案

SwiftTrans 提供免費版和 Pro 版兩種方案：

  • 免費版 - 存取核心翻譯功能和基本特性
  • Pro 版 - 解鎖進階功能，包括無限歷史記錄、優先支援等

可隨時在應用程式內的訂閱檢視中升級到 Pro 版。

設定與自訂

外觀

  • 主題 - 在淺色、深色或跟隨系統外觀之間選擇
  • 語言 - 更改應用程式介面語言

行為

  • 關閉操作 - 選擇關閉視窗時的行為（最小化到選單列、結束或每次詢問）

隱私與安全

  • API 金鑰 - 如果使用自訂 API，您的金鑰將安全儲存在 macOS 鑰匙圈中
  • 本機儲存 - 翻譯歷史本機儲存在您的 Mac 上
  • 更新 - 啟用自動更新檢查以保持最新版本

使用技巧

  • 使用 ⌘+C+C 從任何應用程式即刻存取 SwiftTrans 的迷你模式
  • 在不同翻譯服務之間切換以比較結果
  • 啟用文字轉語音功能，聆聽翻譯的發音
  • 收藏常用語言對以便快速存取
