Política de Privacidad

Última actualización: Enero 2026

Introducción

En SwiftTrans, priorizamos tu privacidad. Esta Política de Privacidad explica cómo manejamos tus datos cuando usas nuestra aplicación de macOS.

Recopilación y Uso de Datos

SwiftTrans está diseñado con la privacidad como prioridad. No recopilamos información personal directamente. Tu historial de traducciones y configuraciones se almacenan localmente en tu dispositivo.

Servicios de Traducción

Para proporcionar funcionalidad de traducción, el texto que envías se envía al proveedor de traducción que selecciones:

  • Google Translate: El texto se envía a la API de traducción de Google. Consulta la Política de Privacidad de Google.
  • Bing Translator: El texto se envía al servicio Bing Translator de Microsoft. Consulta la Declaración de Privacidad de Microsoft.
  • OpenAI / APIs Personalizadas: Si configuras OpenAI u otros puntos finales de API, los datos se envían a esos servicios según tu configuración. Consulta la política de privacidad del proveedor de API respectivo (por ejemplo, Política de Privacidad de OpenAI).

SwiftTrans no almacena tu texto enviado en ningún servidor externo aparte de la transmisión necesaria a estos proveedores de terceros con el propósito de traducción.

Almacenamiento Local

  • Historial de Traducciones: Almacenado localmente en tu Mac. Puedes borrar este historial en cualquier momento dentro de la configuración de la aplicación.
  • Claves API: Si proporcionas tus propias claves API (por ejemplo, para OpenAI), se almacenan de forma segura en tu Llavero de macOS.
  • Preferencias: La configuración de la aplicación (tema, idioma, etc.) se almacena en los Valores Predeterminados de Usuario estándar de macOS.

Informes de Fallos

Podemos recopilar registros de fallos anónimos para ayudarnos a mejorar la estabilidad de la aplicación. Estos registros no contienen datos personales de usuario ni contenido de traducción.
