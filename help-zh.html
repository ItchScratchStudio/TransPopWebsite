帮助与用户指南

概述

SwiftTrans 是您在 macOS 上的优雅翻译伴侣。无论是阅读、写作还是工作时需要快速翻译，SwiftTrans 都能无缝融入您的工作流程。

快速开始

安装

从 Mac App Store 下载 SwiftTrans 并启动。应用将出现在菜单栏中，您可以随时访问。

首次翻译

只需将文本粘贴或输入到输入框中，选择目标语言，SwiftTrans 即刻为您翻译。

主要功能

翻译服务

SwiftTrans 支持多种翻译服务，为您提供最佳结果：

  • Google 翻译 - 快速准确的翻译，支持自动语言检测
  • 必应翻译 - 微软强大的翻译引擎
  • OpenAI 与自定义 API - 连接您自己的 API，使用 AI 模型进行高级翻译

在设置中切换翻译服务，找到最适合您需求的选项。

迷你模式

使用 SwiftTrans 的小巧迷你模式进行快速翻译，不会打断您的工作。非常适合在阅读或写作时翻译短文本。

  • 使用全局键盘快捷键 ⌘+C+C 触发迷你模式
  • 轻量级，随时可用
  • 需要时可保持在其他窗口上方

翻译历史

所有翻译都会自动保存到历史记录中，方便您查找和重用之前的翻译。

  • 搜索您的翻译历史
  • 即时复制过去的翻译
  • 在设置中设置保存项目的最大数量（默认：100）
  • 随时清除历史记录以保护隐私

订阅计划

SwiftTrans 提供免费版和 Pro 版两种方案：

  • 免费版 - 访问核心翻译功能和基本特性
  • Pro 版 - 解锁高级功能，包括无限历史记录、优先支持等

可随时在应用内的订阅视图中升级到 Pro 版。

设置与自定义

外观

  • 主题 - 在浅色、深色或跟随系统外观之间选择
  • 语言 - 更改应用界面语言

行为

  • 关闭操作 - 选择关闭窗口时的行为（最小化到菜单栏、退出或每次询问）

隐私与安全

  • API 密钥 - 如果使用自定义 API，您的密钥将安全存储在 macOS 钥匙串中
  • 本地存储 - 翻译历史本地存储在您的 Mac 上
  • 更新 - 启用自动更新检查以保持最新版本

使用技巧

  • 使用 ⌘+C+C 从任何应用即刻访问 SwiftTrans 的迷你模式
  • 在不同翻译服务之间切换以比较结果
  • 启用文本转语音功能，聆听翻译的发音
  • 收藏常用语言对以便快速访问
